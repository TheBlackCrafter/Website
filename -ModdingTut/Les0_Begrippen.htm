Java Een programeer taal. MineCrat is een van de programma's die in java is gemaakt.
compilen het omschreiven van leesbare programeertaal naar taal die de computer kan begrijpen.
De gecompilede code is veel makkelijker te begrijpen voor de computer en dus is er veel minder lag.
Decompilen het omschreiven van onleesbare computertaal naar leesbare programeer taal.
Minecraft Forge Een mod voor minecraft die zorgt dat meerdere mods samen kunnen werken. Als er bijvoorbeeld twee mods zijn die allebei tin toevoegen, dan kan je de verschillende soorten tin in allebei de recepten gebruiken.
Forge zorgt er ook voor dat wij geen Base-files hoeven te veranderen, zodat minecraft optimaal blijft lopen als er verschillende mods gebruikt worden.
Wij gaan onze mod Forge-Based maken, dat betekent dat je Forge nodig hebt om de mod te kunnen gebruiken
Eclipse Een programma waar je makkelijk java bestanden kan bewerken.
Je kan ook andere programma's zoals notepad++, maar dat is veel onhandigger, Minecraft is namelijk gemaakt voor eclipse.
