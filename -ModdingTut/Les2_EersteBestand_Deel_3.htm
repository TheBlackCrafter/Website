Dan ziet je MOD_Main.java er als je het goed hebt gedaan zo uit.
De plekken die ik geel heb gemaakt moet je nog met wat eigen text invullen. hieronder staat een tabelletje met wat je kan invullen.

nu in code moet verandert worden in
[Naam] De naam van je blokje in de code. Overal waar [Naam] moet het zelfde woord staan.
!Er mag geen spatie in deze naam zitten! ik gebruik voor deze tutorial "VanadiumOre"
[In Game Naam] De naam die je in de game ziet. ik gebruik voor deze tutorial "vanadium ore"
[Block ID] het Block id van het blokje. je kan Hier zien wele block-id's vrij zijn. ik gebruik nu "3940"
[AfbeeldingID] zet hier nu maar 1 neer, later ga ik uitleggen wat dit is.
[Loop Over Geluid] Het geluid dat je hoort als je over het blokje heen loopt en het geluid als je het blokje kapot maakt. Je kan kiezen uit 'soundWoodFootstep','soundGravelFootstep','soundGrassFootstep',
soundStoneFootstep','soundMetalFootstep','soundGlassFootstep','soundClothFootstep',
'soundSandFootstep'.
je moet er wel nog 'Block.' voorzetten. ik gebruik nu "Block.soundSandFootstep"
[Hardness] Hoe hard het blokje is, hoelang het duurt om het blokje kapot te maken. Ik gebruik nu 1.5F
[Resistance] Hoe goed het blokje tegen TNT kan. Ik gebruik nu 10F

Het leukste is om zelf een beetje uit te proberen wat goed bij jouw blokje past. Als het het bestandje 'common/net.minecraft.src/Block.java' in eclipse opent, zie je van elk vanilla blokje de eigenschappen zoals wij die net hebben ingevoert.
Bij mij ziet MOD_Main.java er nu zo uit.
Nu moet je een apart .java bestandje maken voor je blokje.
Dat doe je door lang boven 'VanadiumOre' te blijven hangen met je muis. Dan klik je op de optie create class. Dan druk je op finish.
Dan plak je dit in het bestandje VanadiumOre: pastebin link
nu moet je nattuurlijk we nog effe [Naam] vervangen door de naam die jij hebt gebruikt.

Nu is je mod klaar!!
Druk op safe en dan op de start knop, en je kunt minecraft spelen met je eigen toegevoegde blokje!!
Als je nog een blokje wil toevoegen hoef je alleen deze tutorial nog een keer te doen.
De core tutorial hoef je maar 1 keer te doen.
