Core!

Hi allemaal,
Nu ga ik jullie laten zien hoe je de core (het kloppende hart) van je mod moet maken.

PAS OP: alles in java is hoofdlettergevoelig!!

Ga in de filemanager naar het mapje common.
Daar klik je met je rechtermuisknop op, en ga je naar New, Package.
Dat zie er zo uit.
Dan krijg je een schermpje waar je een naam voor dat package moet invoeren.
Ik noem die nu gewoon MOD.
Het is het makkelijkst als je dezelfde namen als ik gebruik.
Dan klik je met je rechtermuisknop op het package "MOD" die we net hebben gemaakt.
Daar kies je "New" en dan "Class".
Ik noem die Mod_Main omdat dat de belangrijkste modfile wordt.
Dat schermpje ziet er dan zo uit.
Dan maken we nog een class file in de MOD package met als naam "CommonProxy".
Dan ziet je file manager er zo uit.
Dan maak in het mapje "src " een nieuw package genaamd "MOD", en daarin een class bestand met als naam "ClientProxy".
Dan zie je als het goed is dit.
Nu gaan we echt programmeren.
Je klikt dubbel op Mod_Main.java om die te openen, en in je workspace delete je alle tekst, en plak je de volgende tekst daarvoor in de plaats:
Pastebin Link
Dan plak je deze tekst in het bestandje CommonProxy:
Pastebin Link

Pastebin Link
Als het kopieren van pastebin niet helemaal werkt, kan je ook onder aan de pagina bij "RAW Paste Data" de code kopieren, dan zitten er geen enter's tussen.
Dan even wat uitleg. In het bestandje Mod_Main.java staat nu bovenaan "package MOD;" dat betekent dat het bestandje Mod_Main.java in het package MOD staat.
Dan staan daaronder staan 7 regeltjes die beginnen met "import". Die ga ik nu niet uitleggen.
Daaronder staat "@Mod(modid = "[naam maker]_[naam mod]", name = "[naam mod]", version = "[versie]")".
Bij [naam maker] moet je je eigen naam neerzetten.
Bij [naam mod] moet je de naam van je mod neerzetten en bij [versie] moet je de versie van je mod neerzetten.
Alles wat daaronder staat zal ik een andere keer uitleggen.

Dit was het voor vandaag!
