Eclipse?!

Hallo allemaal,

Vandaag ga ik jullie laten zien hoe eclipse werkt en een kleine inleiding geven.
Het eerste wat je moet doen is Eclipes opstarten.
Je krijgt dan als het goed is dit schermpje.
Je gaat dan met het knopje "Browse..." naar de plek waar je MCP hebt geïnstalleerd, en in dat mapje weer naar het mapje "eclips". Zoals je hier ziet.
Het is het handigste voor later als je dat vinkje uit laat.

Nadat Eclipse helemaal is opgestart, krijg je dit scherm te zien.
In dit plaatje zie je wat alles is.
Dan klik je op het pijltje in de file manager links naast "Minecraft". Dan verscijnt er een lijst met onderandere "src", "common" en nog wat andere mappen.
Dan klik je op de pijltjes naast src en common. Dan zie je als het goed is dit.

Dit was hopelijk nog te volgen.
We gaan straks werken in de 2 mappen src en common.
In die mappen zitten zo genoemde "packages". In die packages zitten java bestanden met een .java uitgang.
Als je één van die packages aanklikt, (weer met dat pijltje) dan zie je dus heel veel java bestanden.
Als je dubbel op een van die java bestanden klikt, dan zie je in je workspace de code van dat java bestandje.
Hier heb ik bijvoorbeeld het bestandje "common/net.minecraft.src/BlockClay.java" geopend.
In dat bestandje staat bijvoorbeeld hoeveel clay er gedropt moet worden als je het een clay block kapot maakt.
We gaan nu het getal 4 veranderen in 100. Dan ziet het bestandje er als het goed is zo uit.
Dan druk je op de "safe knop", en dan op de "start-knop".
Dan wordt minecraft afgespeeld met de code die je net veranderd hebt.
Als je dan een wereld aanmaakt en clay mijnt, (in survival) dan krijg je 100 clay!!
GEFELICITEERD!!
Je hebt nu je eerste modification gemaakt!!! Maar dit is NIET een echte forge mod.

In de volgende les gaan we een echte mod maken.
