MCP + Eclipse Setup

Je hebt waarschijnlik net de titel gelezen, en twee vreemde woorden gezien.
Maak je niet druk, het wordt na verloop van tijd allemaal duidelijk!

Minecraft is geschreven in de taal "java".
Nadat de maker van Minecraft de code gemaakt heeft, heeft hij de code "gecompiled".
Het probleem van die computer-code is dat wij als mensen die niet kunnen lezen.
Daarom moet je die computer-coden weer "decompilen".
Daarvoor heb je een programmatje nodig.
Dat programmatje heet MinecraftCoderPack (MCP)

We gaan nu MCP klaarmaken voor gebruik.
Daarvoor moeten we eerst naar de download site van MCP gaan.
Klik hier voor een link
je zoekt op die pagina naar "You can find MCP here: mcp***.zip" .
het bestandje "mcp[versie].zip" moet je downloaden.
Dat ga ik niet uitleggen, jullie als "de niewe jeugd" moet toch weten hoe je iets moet downloaden.
Je krijgt dan een ".zip" bestandje, en die moet je uitpakken in een lege map.

Het volgende wat je moet downloaden is "Minecraft_Server.jar".
Dat kan je doen op de officiele minecraft website.
en hier weer een link voor luie mensen
Je moet de "minecraft_server.jar" hebben, NIET de .EXE!!!

Nu we dit klaar hebben,moeten we forge downloaden.
Forge hebben we nodig omdat we onze mod forge-based gaan maken
Je kan forge hier downloaden.
Nu effe opletten:
Je moet de forge versie hebben die matcht met je minecraft versie: bij mij "1.4.5"
Je moet de laatste versie hebben die bij jouw Minecraft versie past: bij mij "6.4.0.394"
Het is handig om de versienummer van forge te onthouden
Dan moet je de src downloaden.

VOORDAT je deze stap gaat doen, moet je een schone en up to date minecraft instance hebben.
Als je niet weet hoe dat moet, ga terug naar de basisschool of ga naar www.youtube.com


Nu ga je naar je ".minecraft" folder.
Je ziet onderandere de mapjes "bin" en "recources".
Die 2 mapjes kopieer je effe.

Nu gaan we alles bij elkaar zetten:
  1. Eerst pak je dus het mapje met het uitgepakte MCP
  2. Dan pak je de Src van Forge uit en stop je in het zelfdde mapje.
  3. Dan pak je je minecraft_server.jar en stop je die in het mapje "jars".
  4. De mapjes "bin" en "recources" kopieer je ook in het mapje "jars
Nu we MCP kaar hebben, gaan we Ecipse donwloaden.
Je kan ook andere edditors gebruiken, maar daarbij kan ik je niet helpen.
klik hier voor een link naar de downlaod pagina
Je moet dan "Eclipse IDE for Java EE Developers" downloaden. Bij mij is dat de bovenste.
Dan nog effe Eclipse installeren en je bent klaar voor de volgende les!
