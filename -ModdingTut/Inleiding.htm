Welkom,
Op deze site ga ik jou uitleggen hoe je je eigen Minecraft Modification kan maken.
Deze Tutorials zijn gemaakt voor mensen die nog nooit geprogrameerd hebben, en dat wel zouden willen leren.
Je moet niet verwachten dat je na deze cursus een "mod" zoals buildcraft,redpower of Equivelent Exchange kan maken, De mod die we gaan maken wordt -zoals ze noemen- "Forge-Based".
Dat betekent dat je als speler forge nodig hebt om het te spelen.
Weet je niet wat forge is?? Ga dan naar de begrippen lijst!!
Aan het eind van de cursus kan je:

  • een eigen blokje toevoegen
  • een eigen zwaard, pickaxe, shovel, enz... toevoegen
  • een eigen ore generation toevoegen
  • een custom texture toevogen
  • een recept toevoegen
  • een smelting recept toevoegen

De dingen die je nodig hebt om te beginnen zijn:

  1. Een Windows computer. Dat kan Windows xp en Windows 7 zijn.
  2. Een gekochte versie van minecraft. Kan ook een gecrackde versie zijn, maar daarbij geef ik geen ondersteuning!!
